致力于打造基于 Spring Cloud 的、开箱即用的企业级微服务框架。

**YOROZUYA** 是日本漫画《银魂》及其衍生作品中的一个组织，承担他人的各种任务和要求。
> “在这样的时代，一个仍旧保留着武士之魂的男人坂田银时与同伴志村新八、神乐经营着万事屋（只要有钱拿什么工作都接），愉快地过着异想天开的生活。”
